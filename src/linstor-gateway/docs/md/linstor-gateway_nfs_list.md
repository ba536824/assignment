## linstor-gateway nfs list

Lists NFS resources

### Synopsis

Lists the NFS resources created with this tool and provides an
overview about the existing LINSTOR resources and service status.

```
linstor-gateway nfs list [flags]
```

### Examples

```
linstor-gateway nfs list
```
